# Arborito demo

Esta rama te enseña Arborito paso a paso: mapa, lecciones, juegos, puntos y modo construcción.

La escribió **Sage** para que empieces sin manuales técnicos.

Ábrela desde **Bosque → Ramas**. Puedes borrarla cuando ya no la necesites; vuelve si haces un reset completo de datos locales (`reseteverything`).
