# Arborito demo

Esta rama te enseña Arborito paso a paso: mapa, lecciones, juegos, puntos y modo construcción.

La escribió **Sage** para que empieces sin manuales técnicos. Ábrela desde **Bosque → Ramas**.

Es una guía embebida de solo lectura: puedes explorarla entera (incluso construcción), pero no guardar cambios ni borrarla. Si quieres editar, usa **Copiar a Mi jardín**. Tras un reset completo de datos locales (`reseteverything`) vuelve a sembrarse sola.
