# Your study garden

Arborito is the **Treesys app** for learning with a map, practice, and games. This demo branch shows the essentials in one sitting.

@section
title: Map and folders
@/section

Picture a **folder tree** on screen:

- Each **folder** is a topic (for example "Welcome" or "Games").
- Each **leaf** is a lesson to read and practice.
- Put the folder **icon** in the `README` with `icon:` so you do not repeat the emoji in the name.

@section
title: The Forest and your progress
@/section

Everything you import or create lives in the **Forest** (Forest menu). This "Arborito demo" branch is an example: you can **delete** it anytime; it returns if you reset all browser data.

By default your **progress** (lessons read, quizzes passed, streak) is saved **on this device**. Exporting a `.arborito` shares **content**, not your personal history.

@section
title: Four pillars (Treesys vision)
@/section

Treesys builds on four pieces you will see in this demo:

1. **Arborito app** — study with map and outline.
2. **Public courses** — branches other authors share.
3. **Games (Arcade)** — mini-games using your own questions.
4. **Python SDK** — for authors who want to generate or validate content outside the app.

You do not need to code to use Arborito. The SDK is optional for advanced authors and teams.
